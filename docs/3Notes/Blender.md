`s + MM` 移动视图

`MM` 旋转视角

`小键盘1 3 7 9` 正交视图切换

`A` 全选

`L` 关联选择

`c + L` 关联菜单

`G` 移动

`S` 缩放

`R` 旋转

`s + D` 复制物体

`X` 删除

`N` 右侧工具栏

`s + N` 编辑模式法线向内

`c + a + Q` 正交三视图

`a + LM` 循环选择

`c + LM`最短路径选择



`a + Z` 透视模式



`E` 挤出

`I` 内插

`c + B` 倒角

`c + R` 环切

`M` 合并

`v` 断开（右键线，细分）

`F` 填充（面，偶数边栅格填充）

`K` 切刀

`c + J` 合并物体

`c + E` 边菜单-桥接

`P` 分离菜单
